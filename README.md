# Portafolio Profesional — Kenneth Solórzano

Portafolio profesional desarrollado para mostrar trayectoria académica, habilidades técnicas y proyectos destacados en desarrollo backend y software.

## 🚀 Tecnologías Utilizadas

- **Core**: React 19, Vite, TypeScript
- **Estilos**: Tailwind CSS v4
- **Enrutamiento y Estado**: React Router DOM v7
- **Animaciones**: Framer Motion
- **Iconos**: Lucide React + Componentes SVG Personalizados
- **Gráficas**: Recharts
- **SEO**: React Helmet Async

## 📁 Estructura del Proyecto

```
src/
├── assets/          # Imágenes y recursos estáticos de proyectos
├── components/
│   ├── icons/       # Iconos SVG de redes sociales
│   ├── layout/      # Navbar, Footer, Sidebar, Progress, ScrollToTop
│   ├── sections/    # Secciones del HomePage (Hero, Education, Skills, Projects, etc.)
│   └── ui/          # Componentes UI reutilizables (Button, Card, Badge, Lightbox, Modales)
├── data/            # Configuración centralizada de proyectos, habilidades y perfil
├── hooks/           # Hooks personalizados (useScrollPosition, useMediaQuery)
├── pages/           # Vistas (HomePage, ProjectPage, NotFoundPage)
├── types/           # Definiciones de tipos TypeScript
└── utils/           # Utilidades y funciones auxiliares
```

## 🛠️ Instalación y Uso Local

```bash
# Clonar el repositorio
git clone https://github.com/ksolo-2022439/Portafolio.git

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build
```
